# Header Container

This will display two groups with `HeaderDropdown` and `Button` elements.
The styling is optimized for a `Header` if the [breakpoint](?path=/docs/design-system-breakpoints--page) is greater than `lg`.

<Story of={HeaderChildrenStories.Default} />

## Import

```tsx
import { HeaderChildren } from "@imgly-labs/imgly-website-components";
```

## Usage

```tsx
<HeaderWrapper>
  <HeaderContainer>
    <HeaderChildren />
  </HeaderContainer>
</HeaderWrapper>
```
