# Header Menu Container

This will display two groups with `HeaderDropdown` and `MenuButton` elements.
The styling is optimized for use in a `HeaderMenu` element.

<Story of={HeaderMenuChildrenStories.Default} />

## Import

```tsx
import { HeaderMenuChildren } from "@imgly-labs/imgly-website-components";
```

## Usage

```tsx
<HeaderWrapper>
  <HeaderMenu>
    <HeaderMenuChildren />
  </HeaderMenu>
</HeaderWrapper>
```
