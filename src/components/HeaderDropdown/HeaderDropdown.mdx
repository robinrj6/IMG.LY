# Header Dropdown

This dropdown can be used in the `HeaderWrapper` and `HeaderMenu` depending on the [breakpoint](?path=/docs/design-system-breakpoints--page).
The design will switch to a vertical layout if the breakpoint is `lg` or below.

<Story of={HeaderDropdownStories.Default} />

## Import

```tsx
import { HeaderDropdown } from "@imgly-labs/imgly-website-components";
```

## Usage

```tsx
<HeaderDropdown {...rest}>
  <ul style={{ padding: "1rem" }}>
    <li>
      <a href="https://img.ly">IMG.LY</a>
    </li>
  </ul>
</HeaderDropdown>
```
