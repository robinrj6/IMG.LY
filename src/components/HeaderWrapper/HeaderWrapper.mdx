# Header Wrapper

A basic wrapper that defines the `width` and `height` of the `Header`.

## Import

```tsx
import { HeaderWrapper } from "@imgly-labs/imgly-website-components";
```

## Usage

```tsx
<HeaderWrapper>
  <HeaderLogo />
  <HeaderContainer>
    <HeaderChildren />
  </HeaderContainer>
  <HeaderCareers />
  <HeaderMenu>
    <HeaderMenuChildren />
  </HeaderMenu>
</HeaderWrapper>
```

> The `HeaderWrapper` is the relative anchor for `HeaderMenu` by default.
